Powerpointless Presentations

Is that Powerpoint-less, or Power-pointless...?

Image copyright Powerpoint Anonymous

The purpose of this project is to get us started with using HTML and CSS via the Bootstrap framework, and to show how much of Powerpoint's functionality we can do for ourselves - for the rest of it, well, we'll just need to learn some Javascript!

Surprise!

Any old HTML content can go in here, subject to a few conditions (such as, you can't have a modal within a modal.) See the Bootstrap documentation for more details.

Quite often there might be a form in a modal dialog (for instance, to make a comment or to log into a website.) We haven't covered forms, since they usually need some "server-side logic" in order to work.

Using Bootstrap's modal components gets you a lot of features for free - not only the styling and the Javascript that makes it work, but also some convenience features. See what happens if you press the 'Esc' button!
